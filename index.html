Image Filter Project

I created a set of 4 different filters to apply on images, I displayed my knowledge of creating image filters by changing values of pixels, identfying positions of pixels with certin colors and adding interactive optons to customize the filter straight on the webpage. This is a link to my published image filter webpage . Image Filter page

This is a link to a code that was used to create my "Border" filter "Border" Filter Code

This code used most of my knowledge of how to create a image. The image filter consisted of code that used an indexer, a way of identifying positions of certain pixels on the image. I also used if statements to make sure the indexer was choosing which certain pixels i wanted to be a color of my choice. At first grabbed the first 150 pixels in the rows and the columns. Then I grabbed the last 150 pixels in the rows and the columns. I set all chosen red, green, blue and alpha pixels and set them to black. This would make it so that when I applied the filter to the image the border around the image would be a solid black color. I decided to incorperate an option code to allow the tester to change the color of the border. I first needed to define my option and add it into the code, then changed the color value of the pixels chosen to R,G,B instead of 0,0,0 to allow the border to be any color chosen on the color chart on the webpage. This filter shows the most of my understanding of how to manipulate and edit pixel data of an image to create an Image Filter.
